## 介绍

[Node百科](https://gitee.com/BigOldWei/node-pedia)是基于GitHub项目[nodejs.dev](https://github.com/nodejs/nodejs.dev)文档翻译而来的中文Node.js教程。

后继博主会补充更多的资料，从而搭建一个Node百科类仓库，欢迎大家参与我的工作！
